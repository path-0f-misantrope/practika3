# Система Управления Заявками (Repair Service)

Простая система для управления заявками на ремонт бытовой техники. Включает в себя Backend на Go (Gin + pgx) и Frontend на чистом HTML/JS.

## Требования
*   **Go** (Golang) версии 1.20 или выше.
*   **PostgreSQL** (сервер базы данных).

## Установка и Запуск

### 1. Настройка Базы Данных

1.  Откройте ваш клиент PostgreSQL (pgAdmin, DBeaver или командную строку `psql`).
2.  Создайте новую базу данных с именем `repair_service`.
    ```sql
    CREATE DATABASE repair_service;
    ```
3.  Выполните скрипт из файла `schema.sql`, чтобы создать необходимые таблицы (`users`, `requests`, `comments`).

### 2. Заполнение Тестовыми Данными

Чтобы войти в систему, вам понадобятся пользователи. Выполните следующие SQL запросы для создания тестовых пользователей (на основе данных из ваших CSV):

```sql
INSERT INTO users (fio, phone, login, password, type) VALUES 
('Трубин Никита Юрьевич', '89210563128', 'kasoo', 'root', 'Менеджер'),
('Мурашов Андрей Юрьевич', '89535078985', 'murashov123', 'qwerty', 'Мастер'),
('Перина Анастасия Денисовна', '89990563748', 'perinaAD', '250519', 'Оператор'),
('Баранова Эмилия Марковна', '89994563841', 'login2', 'pass2', 'Заказчик');
```

### 3. Настройка Бекена

Откройте файл `main.go`. На строке ~1050 найдите строку подключения к базе данных:

```go
pool, err := pgxpool.New(ctx, "postgres://postgres:zxcqwe123@localhost:5432/repair_service")
```

Убедитесь, что:
*   `postgres` - это ваше имя пользователя в Postgres.
*   `zxcqwe123` - это ваш пароль от Postgres (замените на свой!).
*   `repair_service` - имя созданной базы данных.

### 4. Запуск Сервера

Откройте терминал в папке с проектом и выполните:

```bash
# Загрузка зависимостей
go mod tidy

# Запуск сервера
go run main.go
```

Вы должны увидеть сообщение: `Сервер запущен на :8080`.

### 5. Запуск Клиента

Просто откройте файл `index.html` в любом современном браузере (Chrome, Firefox, Edge).
